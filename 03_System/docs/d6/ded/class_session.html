static flash (   $name,

)
static

Eine Hilfs-Funktion um Session-Variablen zu flashen. Dabei wird beim ersten aufruf eine Session-Variable mit dem Namen $name und der Nachricht $message (optional) gesetzt. Beim zweiten aufruf wird die Session-Variable zurückgegeben und gelöscht. Voraussetzung: Zwei identische Aufrufe mit den gleichen Parametern

Parameter
string$nameName der Session-Variable
string/numeric$message Nachricht die geflasht werden soll.
Rückgabe
string/numeric falls die Session-Variable gesetzt wurde, ansonsten void
