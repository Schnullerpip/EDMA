Description of Parser

Parset die Datei mit RegularExpression. Ergebnis ist ein Array mit Groesse 7: [0]: Array mit allen gefundenen Zeilen (komplett) [1]: Array mit allen Metanamen (ohne _TYP) [2]: Array mit dem Typ (n, s oder d) [3]: Array mit allen Metawerten (gemischt) [4]: Array mit allen Metawerten von Typ String [5]: Array mit allen Metawerten von Typ Datum [6]: Array mit allen Metawerten von Typ Nummer
