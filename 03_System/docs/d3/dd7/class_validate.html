Ausführliche Beschreibung

Eine Klasse zum Überprüfen von Eingabefeldern.

Es kann z.B. definiert werden, dass ein Input-Feld mit dem Namen 'user' mindestens 5 Zeichen haben soll. Oder dass das Passwort Feld required ist etc. Dabei wird ein array mit den Feldern angelegt, welche wiederrum ein Array mit den kriterien haben. Das Array für ein required 'passwort'-Feld sieht z.B. so aus: 'password' => array('required' => true)

Es kann auch ein Ausgabewert für den Feldnamen angegeben werden, wenn die Validierung des Feldes fehlschlägt, z.B.: 'fieldname' => 'Passwort'
