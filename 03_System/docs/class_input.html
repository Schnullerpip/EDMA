Hilfs-Klasse um Input-Felder auf eventuelle Eingaben zu prüfen.

Autor
Sandro

Dokumentation der Elementfunktionen

static Input::exists (   $type = 'post')
static

Prüft ob in dem vom User übermittelten Formular etwas eingetragen wurde.

Parameter
string$typedie Art wie das Formular abgeschickt wird. post oder get
Rückgabe
boolean true, falls der User etwas per post oder get in einem Formular abgeschickt hat. False, falls der $type unbekannt oder keine Eingaben gemacht wurden.
static Input::get (   $item)
static

Holt den Wert von dem Input-Feld mit dem name="$item", falls dieser existiert.

Parameter
string$itemName des Input-Feldes
Rückgabe
string Wert des Input-Feldes oder leerer String, falls der Wert nicht existiert.
static Input::itemExists (   $item)
static

Prüft ob ein Input-Feld mit dem Namen $item übermittelt wurde. Nützlich um die existens von Feldern zu prüfen, die keinen Inhalt haben. Zum Beispiel kann get() bei Buttons vom Typ submit nicht verwendet werden, da sie einen leeren Wert haben.

Parameter
type$itemName des Input-Feldes
Rückgabe
boolean true wenn existiert ansonsten false
